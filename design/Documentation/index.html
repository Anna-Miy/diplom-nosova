A) Folder Structure - top

  • fonts → contains the fonts that are already integrated in the theme
  • images → contains all the layout images and demo pictures
    • dummy → contains placeholder pictures for demonstration purpose
    • minimal → contains images that are used by the minimal page layout option
  • javascripts → contains the jQuery Framework, jQuery plugins and theme specific jQuery code
  • plugins → contains additional jQuery plugins that are including files of different document types
  • stylesheets → containts the main CSS file style.css and more to control fonts and colors of the theme

B) CSS Files and Structure - top

Code Structure

The CSS code structure is pretty simple. In every CSS file you will find headlines that tell you what the following code is about.

Technical Note: In front of every headline is the word @group. This is a special tag, that is used by some CSS-Editors in order to let you navigate through your code faster by the use of a tree view where every group gets displayed as a folder.

File Structure

  • color_settings.css → Lets you control the colors of the layouts for different parts of the template
  • font_settings.css → Lets you control the fonts for 4 different parts of the layout
  • font_setup.css → Imports the Fonts from the /fonts folder with the @font-face method
  • reset.css → CSS reset
  • style.css → This is the main CSS file of the template that contains all the styles, except for colors and fonts.

Note: It's not necessary to include every single CSS file in your <head>. The style.css imports them already, so you just have to link to this file.

C) JavaScript and jQuery - top

This template makes use of the jQuery JavaScript Framework and contains plugins that are written with the frameworks syntax.

All JavaScript files are included in the head of the template files.
Most of the JavaScript files can be found in the javascript folder. The files of plugins that require more than just a JavaScript file are located in the plugins folder.

Plugin Settings and Custom JavaScript

To change the settings of a plugin (e.g. you want to change the transition effects of a slider), edit the script.js file inside the javascript folder. Here you will find all the code that configures and invokes the plugins.

E) Layout Adjustments and Content Input - top

Before or after putting your own content in the template, please make sure you followed these steps:

The Little Things

  • Change the H1 heading to your page or company name.
  • Change the title of your pages.
  • Add a Meta-Description to your pages.
  • Add your own navigation links.
  • Add your own slogan to the Top Bar or delete the whole thing, if you don’t need it.
  • Put content into the Sidebar.
  • Edit the Social Links in the Footer and the Contact Page.
  • Edit the Footer.

Changing the Logo

To change the logo, just overwrite the logo.png file inside the images folder.
If your logo is too long and gets cut off on the right side, you can change the width of the logo space in the style.css file under the Heading Layout Structure > Header > “header.main .wrapper #logo a”.

Changing the Color

You can change the color through the color_settings.css file in the folder stylesheets. To pick colors you can either use the Customizer in the templates Live Demo or a simple Color Picker.

Changing Fonts

Integrated and System Fonts

In the file font_settings.css in the folder stylesheets you can find a list of integrated and system font-families. You can just copy and paste them in the CSS code to change the font families for certain areas of the template.

Google Web Fonts

To set a Google Web Font choose one out of the Google Font Directory. Click on it, select the tab Use this font and follow the instructions. You can then set the font in the file font_settings.css in the folder stylesheets.

Integrating New Fonts

The template is using the @font-face CSS method to integrate web fonts. The easiest way to find fonts is the website Font Squirrel. To integrate a font from this website, please follow these steps:

  • Click on a font that you like
  • Click on License to check if you're allowed to use the font for your purpose
  • Click on @font-face Kit and click Download @font-face Kit.
  • Unzip the file you get
  • Move the folder you get into the template folder fonts
  • Open the file stylesheet.css in the font folder you just moved and copy all of it's code
  • Open the file font_setup.css in the folder stylesheets and paste the code on the end
  • Change the paths to the font files in the code to the font folder in which you moved your font files in
  • Copy the font-family name of the font you want to use and paste as font-family value in the file font_settings.css
  • Enjoy the font
font_setup.css
...
@font-face {
  font-family: 'Sansation';
  src: url('../fonts/sansation/Sansation_Light-webfont.eot');
  src: url('../fonts/sansation/Sansation_Light-webfont.eot?iefix') format('eot'),
       url('../fonts/sansation/Sansation_Light-webfont.woff') format('woff'),
       url('../fonts/sansation/Sansation_Light-webfont.ttf') format('truetype'),
       url('../fonts/sansation/Sansation_Light-webfont.svg#webfontfyQmQF30') format('svg');
  font-weight: normal;
  font-style: normal;
}

Nivo Slider Settings

To set certain options for the Nivo Slider, open the file javascripts/script.js and scroll to the headline Nivo Slider. Here you can set Transition Effect, Animation Speed (animSpeed) and Pause Time (pauseTime) and many more. Please note that Animation Speed and Pause Time need to be adjusted in milliseconds (3000 milliseconds = 3 seconds and 500 milliseconds = 0.5 seconds).

Here are the effects you can insert:

  • sliceDown
  • sliceDownLeft
  • sliceUp
  • sliceUpLeft
  • sliceUpDown
  • sliceUpDownLeft
  • fold
  • fade
  • random

To learn more about the options visit the Nivo Slider creators website: nivo.dev7studios.com.

Slides Slider Settings

To set certain options for the Slides Slider, open the file javascripts/script.js and scroll to the headline Slides Slider. Here you can set Transition Effect, Animation Speed (pause) and Pause Time (play) and many more. Please note that Animation Speed and Pause Time need to be adjusted in milliseconds (3000 milliseconds = 3 seconds and 500 milliseconds = 0.5 seconds).

To learn more about the options visit the Slides Slider creators website: slidesjs.com.

Piecemaker Slider Settings

The files of the Piecemaker Slider are loacated in the directory plugins/piecemaker. You can set options for the slider in the file piecemaker.xml. Bacause of its massive capabilities I suggest that you read the Piecemaker Documentation.

Receive E-Mails from the Contact Form

This template includes a working AJAX-Contact-Form. In order to receive E-Mails that get sent from this form, you need to host the template on a server with PHP 5 and mail support (99% of all hosters should cover these requirements).
All you have to do then, is to open the file contactform_mailer.php and change the value of the receiver variable to your own E-Mail-Address: 

$receiver = "youraddy@example.com";

In this file you can also change the output-notifications of the mailer-script.

Layout Options

You can control how the layout should look by adding certain classes to the body element, like this: 

<body class="minimal simple-nav no-topbar-shine">

This example would render the page with a minimalistic layout, a simple link navigation (instead of buttons) and no glassy shine for the Top Bar (where you can put in your slogan).

Here is an overview of all the classes you can include:

Class Description
with-topbar-content Should be set on every page where you want to include much content or a slideshow in the Top Bar like on the Frontpage
fullwidth Makes room for content in fullwidth of the page and hides the Sidebar.
minimal Lets the page appear more minimalistic and 2-dimensional. Some details will disappear.
no-rounded-corners Disables rounded corners.
no-sliding-links Disables the sliding effect of the sidebar and footer navigation links
no-welcomebar Hides the the Welcome Bar on the Frontpage in which you can put a heading and a slogan.
Note: You can also just delete section#welcomebar to get rid of it.
no-header-top-border Disables the border on top of the header
button-nav Displays the navigation menu links as buttons.
Note: The classes button-nav or simple-nav need to be set on every page to make the navigation menu work correctly.
simple-nav Displays the navigation menu links in a simple way (no buttons).
Note: The classes button-nav or simple-nav need to be set on every page to make the navigation menu work correctly.
no-topbar Hides the the Top Bar in which you can put a slogan.
Note: You can also just delete section#topbar to get rid of it.
no-topbar-shine Disables the shine in the Top Bar
show-header-bottom-border Displays a boder under the header.
Note: This only works if the Top Bar (no-topbar) is disabled.
no-footer Hides the Footer.
Note: You can also just delete footer.main to get rid of it.
no-footer-top-border Disables the big border on top of the footer
no-footerbar Hides the Footer Bar.
Note: You can also just delete section#footerbar to get rid of it.

Lightbox

To let a linked image or Flash-Video appear in a Lightbox, add the attribut class=”lightbox” to the anker element. 

<a class='lightbox' href="big_picture.jpg">
  <img src="picture.jpg" />
</a>

In order to let the user browse through many images inside the Lightbox, assign a sub-attribute to the lightbox value. 

<a class="lightbox[group]" href="big_picture1.jpg"><img src="picture1.jpg" /></a>
<a class="lightbox[group]" href="big_picture2.jpg"><img src="picture2.jpg" /></a>
<a class="lightbox[group]" href="big_picture3.jpg"><img src="picture3.jpg" /></a>

G) Flash-Videos - top

There are two ways to include Flash-Videos. Either you link it with the help of a Lightbox to play it, or you embed it directly into the page.

Lightbox

All what you need, in order to play the video, is the URL. If you want to link to a YouTube-Video, just go to the video of your choice, click below on Share and copy the URL into your anker element. 

<a class="lightbox" href="http://www.youtube.com/watch?v=5L1wHSl_h8w">
  <img src="picture.jpg" />
</a>

It works the same way with videos from vimeo.com.

Embedding

To embed a Video from YouTube or Vimeo just go to the video page, click the share button, make some adjustments (e.g. 613 pixel width on pages with a sidebar or 960 pixel width on fullwidth pages), copy the source code and include it into your page.
You can also embed Flash-Videos directly on the Blog index (just like pictures): 

...
<div class="entry-featured-content">
  <div class="fancy framed with-shadow">
    <iframe src="http://player.vimeo.com/video/16369165" width="613" height="345" frameborder="0"></iframe>
  </div>
</div>
...
